# 消费观念在浪费我的时间

## 我需要确定一个正确的消费观

### 父母的消费观和我的消费观在互相打架

我的父母是一对淳朴的农民。他们的思想一直都是能用就行。对于他们自己，可以说是能省就省。但我知道，他们希望我能有更好的，但他们受制于传统的思想束缚和父母对于孩子的愧疚。所以，他们想要让我尽量节省，但又处于愧疚。只能对我说，你想买就买。
我能看得出来，他们生活的很简单，不需要什么别的。但我是一个比较要强的人，我希望能有更好的，能去和别人比较。至少说不落下风吧。、

### 我经常纠结如何买东西，特别是高价值的东西
我想买什么东西，我首先想到的是看评测。我希望买到一个性价比最高的东西。这是人之常情，我可以理解这样的做法，但我明显有着对于大品牌的盲目信任（我不认为是坏事）。这样的情况，让我很不舒服。我有能力买的品牌，但我因为性价比，而去搜来搜去。这样的行为浪费我大量的时间和精力，我觉得很不舒服。

## **我要坚持自己的消费观**
我的消费观就是，越常用的东西，就用更好的。
比如：1、手机
    2、日常用品（洗发水、洗衣粉）
    3、耳机
    4、衣服
    5、食品（健康最无价）
## **结论：这个618我要买手机和耳机（17和airpods pro2）**
