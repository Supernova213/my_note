# 26年安徽省考反思

## 行测
1. 考前准备：
    这次的考试没有多少准备，可以说除了偶尔在手机小程序上每天坚持刷几题，没有其他准备。\n\n
2. 考中：
    在考试中，我首先选择对图形推理开工，趁着没开始考试，先看着题。尽管看了有一会，但思路仍没有打开。这里可以认为是浪费了些时间，因为我有机会看后面的简单题目。后来，我才发现后面的题目更简单。第二部分就是资料分析，这次找数很快，而且运算都是估算。对比之前，进步是很大的。接下来的答题顺序，是言语、判断、政治、常识、数量。政治我选择d很多，数量我是全选b后再进行补充。
3. 考后总结
    这次我考得很顺利，不会的很快就跳过，是一个不小的进步。
## 申论
1. 考前准备：
    主要是看一些往年试卷，试着理解思路，和解答方向。
2. 考中：
    1. 阅读速度还是不快，会被一些次要元素吸引。
    2. 组织答案有些重复，全部按照阅读顺序往下写。
    3. 公文写作和大作文是我的最大问题，组织言语太耗时，或者说是无法决定一个好的答案。导致最后的大作文，基本都是照抄材料;公文也是差不多，总结性语言太少，估计得分不会高。
3. 考后总结
    以下方面需要提升
    1. 阅读速度训练
    2. 素材积累
    3. 语言组织
